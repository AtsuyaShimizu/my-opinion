# コンセプト
自分の学歴や、職業、性別、支持政党などの情報を各ユーザがクリアにした状態でコミュニケーションを行うSNS
ざっくりとした年収帯もあると嬉しい正確に把握できるかわからない。嘘つけるならいらない。
まずは日本国内をターゲットとする。

## 基本となる考え
現在の社会では分断が進んでいる。その一つの原因がエコーチェンバーに自分が入ってしまっていることを自覚できず、それぞれの考えが先鋭化してしまうことにある。
そこで、それぞれが自らの属性やスタンスを公開した上で議論できるSNSを作ることで、無数にあるエコーチェンバーの中でどのような位置に存在しているのか、また、自分と相違の大きい意見はどのような属性やスタンスからもたらされるものなのかを明確にし、そこから自発的に抜け出すことができるようにしたい。

# 欲しい機能
## タイムライン
自分がフォローしているユーザの投稿が時系列に表示されるタイムライン
リツイート的な機能も欲しい。
各投稿にはユーザの情報とその内容、右下に小さめにメタデータ(投稿日時)が表示される。

### 投稿
投稿には一定の文字数制限を設ける。まずは200字
投稿には返信が可能
投稿には、グッドとバッドの評価が可能。最近のSNSでは外部にバッド数が表示されないことが多い。この辺りが精査が必要。
分析機能があり、その投稿をクリックするなどすると、どのような属性やスタンスの人がその投稿を見ているのか、好意的/否定的なのかをグラフなどで視覚的に見ることができる。

## ユーザ画面
各ユーザ個人の画面自分や他人のプロフィールや投稿の一覧が表示される画面
その人の情報を知ることができる。
将来的には最大500字程度でその人の性格やスタンス、属性を端的にAIでまとめた「分析」項目があるといい。

### ユーザ情報
ユーザは「ユーザ名」、「ユーザID」を持つ。ユーザIDは固有の値である必要がある。
ユーザは自由にアイコンを設定可能。
ログイン用にパスワードも必要。
